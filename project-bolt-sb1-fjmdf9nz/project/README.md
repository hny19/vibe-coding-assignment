# Smart Todo List App

This project was developed using Bolt.new, a vibe coding tool, as part of the "Exploring Vibe Coding Experience" assignment.

![Screan shots](ekran-goruntusu.png)

## About the Project
This application is a modern todo list that allows users to organize their daily tasks. The entire project was generated through AI prompts (Vibe Coding) without writing manual code.

## Features
- Task Management: Add, edit, and delete tasks.
- Status Tracking: Mark tasks as completed.
- Persistence: Data is saved to LocalStorage so it is not lost on refresh.
- Filtering: Filter tasks by All, Active, and Completed status.
- Responsive Design: Works on both mobile and desktop devices.

## Technologies Used
- Vibe Coding Tool: Bolt.new (StackBlitz AI)
- Framework: React
- Build Tool: Vite
- Styling: Tailwind CSS

## How to Run
To run this project locally, follow these steps:

1. Download or clone the repository.
2. Open a terminal in the project folder.
3. Install dependencies:
   npm install
4. Start the application:
   npm run dev
5. Open your browser and navigate to the link shown in the terminal (usually http://localhost:5173).
